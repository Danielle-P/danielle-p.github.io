adventure.gen app

Project & Idea

My most recent side project was to create the design for a narrative adventure game app, which uses AI to generate an illustrated narrative based on user inputs (the idea was based on "choose your own adventure" books). The game is currently available on the App Store, and I am working with the developer to iterate on the design and game features based on feedback.

My priorities when creating these designs were:

  • simplicity: we wanted the UI to be really simple so as not to distract from the story, and to allow for a more immersive experience.
  • flexibility: the UI had to be really flexible, to accommodate different imagery. We decided on 7 adventure themes to start with, from desert to space. I designed a subtle UI for each theme.
